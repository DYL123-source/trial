如何安装Oxygen XML Editor？

途径一：

直接从官网（https://www.oxygenxml.com/）下载官方最新版软件（官网下载的软件只能免费试用30天）

安装+破解步骤：

  1. 下载软件压缩包文件，首先选择与电脑系统相符的文件夹，然后点击“oxygen-64bit.exe”安装原版程序

    P

  2. 选择语言并确认，阅读并同意软件安装协议。

  3. 选择软件安装路径。

  4. 选择Oxygen XML Editor安装组件，默认安装即可。

  5. 选择附加任务，建议创建桌面快捷方式。

  6. 等待程序安装完毕，直接点击【Finish】按钮退出向导即可。

  7. 破解办法：将文件中的“oxygen.jar”复制到软件根目录下lib文件夹中，选择替换同名文件。 默认路径为：C:\Program Files\Oxygen XML Editor 21\lib

  8. 启动程序，软件将自动弹出注册框，选择use a license key ，将下面的秘钥粘贴到红色区域内，点【ok】即可。
