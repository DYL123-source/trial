如何利用Oxygen XML Editor进行各个模块的撰写？

下面将展示单一模块的撰写操作：

  1. 首先点击【Project】新建一个项目。注意项目名称和存储路径。

  2. 点击菜单栏中的【File】，并选择【New】。

  3. 输入想要撰写的模块文件类型，如【task】文件。编写【Title】并选择存储文件夹。新建新的文件。

  4. 在Author视图下，根据提示，输入相关内容即可。

    按【回车键】可以选择添加不同种类的内容。

  5. 按照上面操作方法，新建并完成剩余类型文件的编写。此时在Project项目栏下可以看到所编写的所有文件。

  6. 通过新建【ditamap】文件，将所编写文件链接起来。

    注意链接子文件要在【DITA Maps Manager】视图下进行。

  7. 选择ditamap文件，右键选择【Append child】，再选择【Reference】，将所有的子文件添加植入到ditamap文件下。

  8. 在DITA Maps Manager视图下，选择ditamap文件，点击右上方工具栏中的扳手图标，选择输出为PDF。

  9. 输出PDF文件。
